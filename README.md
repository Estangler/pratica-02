# Pokédex React + TypeScript

Uma aplicação Pokédex desenvolvida com **React** e **TypeScript** consumindo dados da [PokéAPI](https://pokeapi.co/).

O projeto foi criado com foco em prática de:

- Consumo de APIs REST
- Componentização
- TypeScript
- Manipulação de estado
- Arquitetura de aplicações React
- Renderização dinâmica
- Tratamento de erros e loading states

---

# ✨ Funcionalidades

- Listagem dos 151 primeiros Pokémons
- Busca por nome
- Filtro por tipo
- Modal com detalhes do Pokémon
- Exibição de:
  - Stats
  - Habilidades
  - Tipos
  - Peso
  - Altura
  - Taxa de captura
- Interface responsiva
- Tratamento de erros
- Estados de carregamento
